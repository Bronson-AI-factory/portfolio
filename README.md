## Сайт‑визитка с AI‑чатботом

Одностраничный сайт‑визитка CEO с встроенным чат‑ботом «Ассистент CEO».  
Чат‑бот консультирует по услугам агентства недвижимости, ценам и может связать посетителя с CEO.

### Структура

- `index.html` — страница визитки + виджет чата
- `api/chat.js` — Vercel serverless API (MiniMax)
- `knowledge.json` — база знаний (услуги, цены, контакты CEO)

### Локальный просмотр

Откройте `index.html` в браузере. Чат будет виден, но ответы AI не появятся без backend.

### Деплой на Vercel

1. Подключите репозиторий к [Vercel](https://vercel.com).
2. В настройках проекта добавьте переменную окружения:
   - `MINIMAX_API_KEY` — ключ с [platform.minimax.io](https://platform.minimax.io)
3. Деплой — Vercel автоматически подхватит `api/` и статику.

После деплоя чат‑бот будет отвечать на вопросы об услугах и ценах и сможет связать посетителя с CEO.
