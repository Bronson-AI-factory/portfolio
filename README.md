## Сайт‑визитка «Алексей Морозов — Frontend‑разработчик»

Одностраничный сайт‑портфолио, сверстанный на чистом HTML и CSS без фреймворков.  
Страница показывает краткий профиль фронтенд‑разработчика, стек технологий, описание подхода к работе и удобные ссылки для связи.

Основной файл проекта:

- `index.html` — полностью самодостаточная страница, все стили и скрипты находятся внутри файла.

### Как открыть локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Bronson-AI-factory/portfolio.git
   cd portfolio
   ```
2. Откройте файл `index.html` двойным кликом или через браузер:
   - перетащите `index.html` в окно браузера  
   - или в самом браузере выберите «File → Open» и укажите этот файл.

Никакой сборки и зависимостей не требуется — достаточно современного браузера.

### Просмотр через GitHub Pages

Репозиторий можно развернуть как GitHub Pages:

1. Войдите в репозиторий на GitHub: `Bronson-AI-factory/portfolio`.
2. Откройте вкладку **Settings → Pages**.
3. В блоке **Build and deployment**:
   - **Source**: выберите **Deploy from a branch**;
   - **Branch**: выберите `main` и папку `/ (root)`.
4. Сохраните настройки и подождите деплоя (обычно 1–2 минуты).

После этого сайт будет доступен по адресу вида:

`https://bronson-ai-factory.github.io/portfolio/`
